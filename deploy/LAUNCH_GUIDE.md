## 三、推荐上线方案

### 方案 A：国内正式上线（推荐）

**架构**：阿里云全家桶
```
用户 → 阿里云 CDN → 阿里云 ECS → Supabase 数据库
                            ↓
                      阿里云 OSS（媒体存储）
```

**所需资源**：
1. 域名（阿里云购买，约 ¥50/年）
2. ICP 备案（必须，约 7-20 天）
3. ECS 服务器（2核4G，约 ¥100/月）
4. CDN 流量（按量付费，约 ¥20-50/月）
5. SSL 证书（免费 Let's Encrypt 或 阿里云免费证书）

**总成本**：约 ¥150-200/月

---

### 方案 B：快速上线（无需备案）

**架构**：Vercel + Cloudflare
```
用户 → Cloudflare CDN → Vercel → Supabase 数据库
```

**所需资源**：
1. 域名（Cloudflare/Namesilo 购买，约 ¥50/年）
2. Cloudflare 免费套餐
3. Vercel 免费套餐

**总成本**：约 ¥50/年（仅域名）

**注意**：未备案域名国内访问可能不稳定

## 四、详细上线步骤

### 第一步：购买域名

1. 访问 [阿里云域名注册](https://wanwang.aliyun.com/domain/)
2. 搜索想要的域名（如 `hudongju.com`）
3. 选择后缀：`.com` 最通用
4. 完成购买和实名认证

**推荐域名**：
- 短剧类：`hudongju.com`、`duanjuapp.com`
- 创作类：`storyeditor.com`、`creater.com`

---

### 第二步：ICP 备案（国内必须）

**备案流程**（约 7-20 天）：
1. 登录 [阿里云 ICP 备案系统](https://beian.aliyun.com/)
2. 填写主体信息（个人/企业）
3. 填写网站信息
4. 上传资料（身份证、域名证书等）
5. 等待审核
6. 完成备案，获取备案号

**备案要求**：
- 域名实名认证通过
- 有国内服务器（ECS）
- 网站内容合法合规

---

### 第三步：购买服务器

**推荐配置**：
| 项目 | 推荐值 | 说明 |
|------|--------|------|
| CPU | 2核 | 够用 |
| 内存 | 4GB | 推荐 8GB |
| 存储 | 40GB SSD | 视媒体量增加 |
| 带宽 | 5Mbps | 按需增加 |
| 地域 | 华东/华北 | 用户集中的区域 |

**购买地址**：[阿里云 ECS](https://ecs.console.aliyun.com/)

---

### 第四步：部署应用

**方式1：手动部署**
```bash
# SSH 连接服务器
ssh root@你的服务器IP

# 运行一键部署脚本
curl -fsSL https://raw.githubusercontent.com/BABAAIMENGMENG/interactive-story-editor/main/deploy/aliyun/deploy.sh | bash
```

**方式2：自动化部署（推荐）**
- 配置 GitHub Actions
- 推送代码自动部署到服务器

---

### 第五步：配置域名和 CDN

**1. 域名解析**
```
类型: A
主机: @
值: 你的服务器IP

类型: CNAME  
主机: www
值: @
```

**2. 配置 CDN**
- 登录 [阿里云 CDN 控制台](https://cdn.console.aliyun.com/)
- 添加加速域名
- 配置源站（你的服务器IP）
- 配置 HTTPS 证书

---

### 第六步：配置 HTTPS

**方式1：Let's Encrypt（免费）**
```bash
# 在服务器上执行
apt install certbot python3-certbot-nginx
certbot --nginx -d yourdomain.com
```

**方式2：阿里云免费证书**
- 控制台 → SSL 证书 → 免费证书
- 申请后下载 Nginx 格式
- 配置到 Nginx
