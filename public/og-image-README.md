# OG 分享图片说明

## 图片要求
- **文件名**: `og-image.png`
- **位置**: `public/og-image.png`
- **尺寸**: 1200 x 630 像素
- **格式**: PNG 或 JPG

## 设计建议
1. 包含平台 Logo 和名称 "VES互动短剧"
2. 展示平台特色（全景、互动、剧情）
3. 使用明亮的颜色和清晰的文字
4. 避免文字太小，在手机上也要清晰可见

## 在线工具推荐
- Canva: https://www.canva.com/
- Figma: https://www.figma.com/
- Adobe Express: https://www.adobe.com/express

## 生成完成后
1. 将图片保存为 `og-image.png`
2. 放到 `public/` 目录下
3. 部署后分享链接时会自动显示
